# Калькулятор биологического возраста (Горелкин–Пинхасов)

Онлайн-калькулятор биологического возраста и скорости старения по антропометрии.
Статическая страница (`index.html`), считает прямо в браузере — без бэкенда и
без отправки данных куда-либо.

**Метод:** патент РФ № 2387374 (Горелкин А.Г., Пинхасов Б.Б.). Коэффициенты —
по научной публикации (подтверждены врачом).

Формула:
- `РЛ` = возраст − 21 (муж) / − 18 (жен)
- `КСС` = (ОТ × Вес) / (ОБ × Рост² × F), рост в метрах
  - `F_муж = 17.2 + 0.31·РЛ + 0.0012·РЛ²`
  - `F_жен = 14.7 + 0.26·РЛ + 0.001·РЛ²`
- `БВ` = КСС × РЛ + 21 (муж) / + 18 (жен)
- КСС: <0.95 замедленное · 0.95–1.05 норма · >1.05 ускоренное старение

> Оценочный инструмент, не заменяет консультацию врача.
